GNSS Guided Autonomous Car

Project information

  • Category: GNSS Guided Autonomous Car
  • Project date: 2025
  • GitHub Repo

GNSS guided robotic car implementation to test the capabilities of low-cost GNSS receiver sensor (u-blox NEO-M8N).

Key features include:
• Fully assembled robotic car controlled with an ESP32
• Multiple sensors for increased accuracy: Magnetometer (as compass), IMU (will be implemented later), Ublox NEO-M8N, buzzer (for 8-bit music demonstration)
• FreeRTOS implementation for concurrency of sensor polling, navigation, commanding and telemetry, music playing
• Modern web interface to view and control the car's location and to monitor the sensor readings • Plays Scat Man with buzzer!
